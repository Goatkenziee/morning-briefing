# Morning Briefing

A daily briefing dashboard that summarizes your unread Gmail, calendar events, Slack alerts, GitHub activity, and more — all in one place.

Built with **Next.js 14**, **TypeScript**, and **Tailwind CSS**.

## Features

- 📨 Email summary (Gmail integration)
- 💬 Slack alerts & platform health
- 🐙 GitHub repo activity
- 📊 Google Ads campaign performance
- 💳 Stripe payment overview
- 📱 Social account status (Instagram, Facebook, Discord)
- ✅ Prioritized action items

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the briefing.

## Integrations

Connect your services in Settings → Integrations:
- Gmail (for emails + calendar)
- Slack (for alerts)
- GitHub (for repo activity)
- Google Ads (for campaign data)
- Stripe (for revenue tracking)
